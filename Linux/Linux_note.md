# Linux 下的文件

| / 下的文件 | 含义           |
| :---:      | :---:          |
| <++>       | <++>           |
| bin        | 可执行文件     |
| boot       | 系统阅读并启动 |
| dev        | 硬件           |
| etc        | 系统配置文件   |
| home       | 家目录         |
| mnt        | U盘，硬盘显示  |
| proc       | 所有进程       |
| root       | 管理员         |
| srv        | 服务文件       |
| sys        | 系统文件夹     |
| tmp        | 临时文件       |
| usr        | 装载软件       |
| var        | 程序的缓存     |
